# Personal Portfolio Website

A responsive personal portfolio website showcasing my projects, technical skills, and professional profile.

## Overview

This repository contains the source code for my portfolio website, built to present my software engineering and machine learning projects in a clean, recruiter-friendly format.

## Features

- Responsive single-page portfolio design
- Featured project showcase with technical highlights
- Technology stack section with visual skill representation
- GitHub Pages deployment for live hosting
- Minimal and professional UI focused on readability

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
